## Development

### Installing Dependencies

From within the root directory:
```sh
npm install
```

If you do not have React-Native CLI installed:
```sh
npm install -g react-native-cli
```

### Simulating the Application in Xcode

From within the root directory:

Build the Database
```sh
npm run startdb
npm run buildb
```

Start the Server
```sh
npm run startServer
```

Launch the App
```sh
npm run startApp
```
